# Platform Engineering Workstation

Reproducible Linux workstation build for platform engineering, homelab operations, cloud tooling, automation, and development.

## Purpose

This repo documents and automates the setup of my Ubuntu-based engineering workstation.

## Structure

- `ADR/` - Architecture Decision Records
- `SOP/` - Standard Operating Procedures
- `Architecture/` - Diagrams and design notes
- `Docs/` - General documentation
- `Bootstrap/` - Setup scripts
- `Dotfiles/` - Config files
- `Scripts/` - Helper scripts
- `Assets/` - Images and supporting files
