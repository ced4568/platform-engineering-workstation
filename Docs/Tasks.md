# Tasks

Common task commands:

- `task health` - full health check
- `task validate` - validate tools
- `task versions` - show tool versions
- `task network:check` - check DNS and GitHub connectivity
- `task security:scan` - scan for obvious secrets
- `task ssh:check` - check SSH config safety
- `task dotfiles:audit` - audit managed dotfiles
- `task configs:save` - save live configs
- `task repos:status` - show local repo status
- `task repos:sync` - sync GitHub repos
- `task repos:duplicates` - check duplicate repo directories
