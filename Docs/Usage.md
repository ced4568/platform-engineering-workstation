# Usage

Common commands for the Platform Engineering Workstation.

## Commands

- Validate workstation: `task validate`
- Update workstation: `task update`
- Sync GitHub repositories: `task repos:sync`
- Check repository status: `task repos:status`
- Back up configs: `task dotfiles:backup`
- Open project launcher: `task open`
- Clean Docker: `task docker:clean`
- Full bootstrap: `task bootstrap`

## Notes

Run commands from the repository root:

`~/Dev/GitHub/ced4568/platform-engineering-workstation`
